# Affix

Affix component makes elements stick to viewport when scrolling.

## Installation

```bash
pnpm add @chemist-ui/core @chemist-ui/react antd
```

## Basic Usage

```tsx
import { Affix, Button } from 'antd';

export default function AffixExample() {
  return (
    <Affix offsetTop={10}>
      <Button type="primary">Affix Button</Button>
    </Affix>
  );
}
```
