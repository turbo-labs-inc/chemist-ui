# List

List component for displaying data.

## Basic Usage

```tsx
import { List } from 'antd';

const data = ['Item 1', 'Item 2', 'Item 3'];

export default function ListExample() {
  return (
    <List
      dataSource={data}
      renderItem={(item) => <List.Item>{item}</List.Item>}
    />
  );
}
```
