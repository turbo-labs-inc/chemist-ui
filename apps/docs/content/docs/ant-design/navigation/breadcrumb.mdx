# Breadcrumb

Display the path to the current page with breadcrumb navigation.

## Basic Usage

```tsx
import { Breadcrumb } from 'antd';

export default function BreadcrumbExample() {
  return (
    <Breadcrumb
      items={[
        { title: 'Home' },
        { title: 'Application' },
        { title: 'Detail' },
      ]}
    />
  );
}
```
