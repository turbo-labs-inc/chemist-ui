# Radio

Radio button for selecting one option from a group.

## Basic Usage

```tsx
import { Radio } from 'antd';

export default function RadioExample() {
  return (
    <Radio.Group>
      <Radio value={1}>Option 1</Radio>
      <Radio value={2}>Option 2</Radio>
    </Radio.Group>
  );
}
```
