# Flex

Flexbox layout component for aligning elements.

## Basic Usage

```tsx
import { Flex, Button } from 'antd';

export default function FlexExample() {
  return (
    <Flex gap="small">
      <Button>Button 1</Button>
      <Button>Button 2</Button>
      <Button>Button 3</Button>
    </Flex>
  );
}
```
