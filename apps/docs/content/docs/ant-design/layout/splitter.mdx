# Splitter

Splitter provides resizable panel layouts with drag handles.

## Installation

```bash
pnpm add @chemist-ui/core @chemist-ui/react antd
```

## Basic Usage

```tsx
import { Splitter } from 'antd';

export default function SplitterExample() {
  return (
    <Splitter>
      <Splitter.Panel>First Panel</Splitter.Panel>
      <Splitter.Panel>Second Panel</Splitter.Panel>
    </Splitter>
  );
}
```
