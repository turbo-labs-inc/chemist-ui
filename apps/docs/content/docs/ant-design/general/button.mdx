title: Button

import { Button, Space } from 'antd';

Buttons allow users to take actions with a single tap or click.

## Installation

```bash
pnpm add @chemist-ui/core @chemist-ui/react antd
```

## API Reference

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `primary \| default \| dashed \| text \| link` | `default` | Button type |
| `size` | `large \| middle \| small` | `middle` | Button size |
| `disabled` | `boolean` | `false` | Disabled state |
| `loading` | `boolean \| { delay: number }` | `false` | Loading state |
| `danger` | `boolean` | `false` | Danger state |
| `block` | `boolean` | `false` | Full width button |
| `ghost` | `boolean` | `false` | Ghost style (transparent background) |
| `shape` | `default \| circle \| round` | `default` | Button shape |
| `icon` | `ReactNode` | - | Icon element |
| `onClick` | `(event) => void` | - | Click handler |
| `htmlType` | `button \| submit \| reset` | `button` | Native button type |

## Accessibility

- Supports keyboard navigation (Enter/Space to activate)
- Proper ARIA attributes automatically applied
- Focus indicators meet WCAG 2.1 standards
- Disabled buttons are not keyboard focusable
