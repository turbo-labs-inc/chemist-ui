# Tree

Display data in a hierarchical tree structure.

## Basic Usage

```tsx
import { Tree } from 'antd';

const treeData = [
  {
    title: 'parent 1',
    key: '0-0',
    children: [
      { title: 'leaf', key: '0-0-0' },
      { title: 'leaf', key: '0-0-1' },
    ],
  },
];

export default function TreeExample() {
  return <Tree treeData={treeData} />;
}
```
