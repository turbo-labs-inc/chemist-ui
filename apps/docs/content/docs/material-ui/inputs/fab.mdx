# Floating Action Button (FAB)

Prominent button for primary actions.

## Basic Usage

```tsx
import { Fab } from '@mui/material';
import AddIcon from '@mui/icons-material/Add';

export default function FabExample() {
  return (
    <Fab color="primary" aria-label="add">
      <AddIcon />
    </Fab>
  );
}
```
