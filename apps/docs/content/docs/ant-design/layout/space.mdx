# Space

Space component provides consistent spacing between child components.

## Installation

```bash
pnpm add @chemist-ui/core @chemist-ui/react antd
```

## Basic Usage

```tsx
import { Space, Button } from 'antd';

export default function SpaceExample() {
  return (
    <Space>
      <Button>Button 1</Button>
      <Button>Button 2</Button>
      <Button>Button 3</Button>
    </Space>
  );
}
```
