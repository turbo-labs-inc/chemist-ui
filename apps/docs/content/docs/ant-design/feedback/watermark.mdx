# Watermark

Add watermarks to protect content.

## Basic Usage

```tsx
import { Watermark } from 'antd';

export default function WatermarkExample() {
  return (
    <Watermark content="Chemist UI">
      <div style={{ height: 500 }}>
        <p>Content protected by watermark</p>
      </div>
    </Watermark>
  );
}
```
